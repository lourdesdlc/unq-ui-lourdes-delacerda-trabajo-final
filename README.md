# UNQ » UIs » TFI » Preguntados

Trabajo Final Integrador para la materia Construcción de Interfaces de Usuario, Universidad Nacional de Quilmes.

## Datos del Alumno

- **Nombre:** Lourdes de la Cerda

## Descripción

Este proyecto es una recreación del juego **Preguntados** desarrollado en **React**. Permite seleccionar dificultades, responder preguntas consumiendo la API externa de la materia y visualizar el puntaje final.

## Instalación y Ejecución

Para correr este proyecto localmente, seguir estos pasos:

1.  **Clonar el repositorio:**

    ```sh
    git clone https://github.com/lourdesdlc/unq-ui-lourdes-delacerda-trabajo-final.git
    ```

2.  **Ingresar a la carpeta:**

    ```sh
    cd unq-ui-lourdes-delacerda-trabajo-final
    ```

3.  **Instalar dependencias:**

    ```sh
    npm install
    ```

4.  **Ejecutar el proyecto:**

    ```sh
    npm run dev
    ```

5.  **Jugar:**
    Abrir en el navegador la URL que muestra la consola (generalmente `http://localhost:5173`).

## Tecnologías Utilizadas

- **React** (Framework principal)
- **Vite** (Build tool)
- **Axios** (Peticiones HTTP a la API)
- **React Router DOM** (Navegación entre pantallas)

## Notas Técnicas

- El proyecto consume la API pública: `https://preguntados-api.vercel.app`
- Se implementó lógica de juego completa (selección, respuesta, puntaje).
